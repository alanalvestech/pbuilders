# Caption — Project Glasswing

## Caption

Google, Apple, Microsoft, AWS e NVIDIA no mesmo projeto. Isso não acontece por acidente. 👇

A Anthropic lançou o Project Glasswing com 12 organizações para usar o Claude Mythos de forma defensiva.

O Mythos é um modelo treinado especificamente para encontrar vulnerabilidades em software. O resultado: ele supera praticamente qualquer especialista humano nessa tarefa. Já encontrou falhas com décadas de existência em sistemas que todo mundo usa.

O argumento da Anthropic é direto: essa capacidade vai se disseminar. Quando qualquer ator mal-intencionado tiver acesso a modelos assim, o cenário de ataques muda. Melhor garantir que a defesa chegue primeiro.

Quando as maiores empresas de tecnologia do mundo param de competir pra se juntar num projeto assim, o sinal é claro: chegamos num nível que ninguém pode ignorar.
