A Anthropic acabou de vazar o código-fonte inteiro do Claude Code. Por acidente. 🚨

Não foi hack. Foi um arquivo de map esquecido no build. Qualquer pessoa conseguiu baixar e ler tudo.

E no código vieram features que eles nunca anunciaram:

→ Kairos: agente autônomo rodando em background, "sempre ligado"
→ Coordinator Mode: Claude controlando múltiplos agentes em paralelo
→ Auto Mode: zero confirmações de permissão, tudo aprovado automaticamente
