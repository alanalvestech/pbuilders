# PBuilders

## Workflow
- Sempre fazer commit e push automaticamente após mudanças no código
- Mensagens de commit em português

## Deploy
- Deploy automático via GitHub Actions ao fazer push na main
- Hospedagem: Cloudflare Pages (projeto: pbuildersai)
- Domínio: pbuilders.ai

## Browser (Chrome MCP)
- Usar `read_page` para extrair referências de elementos
- Usar `find` para localizar elementos por descrição
- Interagir usando `ref`, não coordenadas
- Evitar screenshots a menos que explicitamente solicitado
